À propos d'AgriScan Pro

AgriScan Pro est une application développée par "Yousra Ameur" pour détecter les maladies des plantes grâce à l'intelligence artificielle.

  • Utilisation de MobileNetV2 avec transfert learning.
  • Framework Flask pour une interface web moderne.
  • Dataset PlantVillage pour l’entraînement.

Projet réalisé dans le cadre du cursus à l'Euro-Mediterranean University of FES.
